# 💰 TransactionApp - Personal Finance Tracker

A modern, full-stack expense tracking application to help you manage your finances effortlessly. Track your income and expenses with a beautiful, responsive interface.

## ✨ Features

### 📱 Frontend Features
- ✅ **Transaction Management** - Add, edit, and delete transactions
- 📊 **Real-time Dashboard** - View total income, expenses, and balance
- 🔍 **Advanced Search & Filters** - Filter by type, category, date range, and amount
- 🌓 **Dark/Light Mode** - Toggle between themes with persistence
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile
- 🎨 **Modern UI** - Glass morphism design with smooth animations
- 💳 **Category Management** - 7+ predefined categories with icons

### 🔧 Backend Features
- 🚀 **RESTful APIs** - Full CRUD operations for transactions
- 🗄️ **Database Integration** - Spring Data JPA with Hibernate
- 🔄 **CORS Configuration** - Secure cross-origin requests
- ✅ **Input Validation** - Server-side validation for data integrity
- 📝 **Error Handling** - Global exception handling with meaningful responses
- 🔍 **Filtering Support** - Query by category, date, and amount range
